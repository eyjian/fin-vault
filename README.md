# FinVault（锦仓）项目README

**项目名称**：FinVault（锦仓）

**项目定位**：个人全资产统一管理工具（金融金库），聚合银行理财、基金、股票等多渠道资产，解决资产分散在银行APP、天天基金、券商、富途牛牛等多平台，难以统一查看、统筹管理的痛点，实现“一处管理，全仓掌控”。项目包含前端、后端多模块，采用单仓（monorepo）设计，统一管理所有代码与开发文档，便于开发、维护与协同。

**项目寓意**：FinVault = Finance（金融） \+ Vault（金库），寓意将分散在各平台的资产，统一收纳进“金融金库”；中文名称「锦仓」，取“锦藏千金，仓纳万资”之意，既体现资产的珍贵与规整，也传递“稳健收纳、统筹增值”的核心价值。

## 📖 项目介绍

随着个人理财渠道的多元化，越来越多的人将资产分散在不同平台：银行APP购买理财产品、天天基金定投基金、券商APP交易A股、富途牛牛操作港美股…… 分散管理不仅导致资产状况混乱、难以快速掌握整体盈亏，也无法高效进行资产配置与规划。

FinVault（锦仓）旨在打造一款轻量、高效、安全的个人全资产管理工具，通过接口对接或手动录入的方式，聚合所有平台的理财、基金、股票资产，实现：

- 全资产统一视图：一眼看清所有平台的资产总额、持仓明细、盈亏情况

- 分类管理：按理财、基金、股票、现金等类型分类，清晰梳理资产结构

- 盈亏统计：自动计算各资产、各平台的收益情况，生成可视化报表

- 资产提醒：理财到期、基金止盈止损、股票行情异动等提醒功能

- 安全保障：本地数据加密存储，不泄露个人敏感信息与账户密码

## 🎯 核心功能规划

### 1\. 资产聚合模块

- 多平台支持：对接银行理财、天天基金、主流券商（东方财富、华泰、中信等）、富途牛牛等平台（优先支持API对接，无API的平台支持手动录入）

- 资产同步：支持手动同步/自动定时同步各平台资产数据，确保数据实时准确

- 资产分类：自动将资产划分为「理财产品」「基金」「股票」「现金及等价物」四大类，支持自定义分类

### 2\. 持仓管理模块

- 明细查看：查看每笔资产的持仓金额、持仓成本、当前市值、盈亏金额、收益率

- 历史记录：记录资产的买入、卖出、赎回、到期等操作记录，支持筛选与搜索

- 持仓分析：按平台、资产类型、时间维度分析持仓占比，辅助资产配置决策

### 3\. 盈亏统计模块

- 实时盈亏：实时计算总盈亏、各平台盈亏、各资产类型盈亏

- 收益报表：生成日/周/月/年收益报表，支持可视化图表（折线图、饼图）展示

- 收益对比：对比不同资产、不同平台的收益表现，辅助优化理财策略

### 4\. 提醒与设置模块

- 到期提醒：理财产品到期、基金定投到期、信用卡还款等提醒

- 行情提醒：股票涨跌预警、基金净值异动提醒，支持自定义预警阈值

- 个性化设置：支持数据加密、同步频率、提醒方式等自定义设置

### 5\. 安全模块

- 本地加密：资产数据本地存储，采用AES加密算法，保护个人敏感信息

- 权限控制：支持设置应用密码、指纹/面容解锁，防止他人查看

- 数据备份：支持本地备份/恢复，避免数据丢失

## 🔧 技术栈规划（单仓多模块，可根据开发实际调整）

### 后端（可选，用于API对接与数据同步）

语言：Python/Go/Java

- 数据同步：定时任务（后端）/ 手动触发（前端）

### 可视化

图表库：ECharts、FlChart（Flutter）、MPAndroidChart（Android原生）

## ⚠️ 注意事项

- 项目为个人开源项目，仅用于个人资产管理，不涉及任何资金交易、账户登录（对接API时仅获取资产查看权限，不获取账户操作权限）

- 数据安全：所有敏感数据（如资产信息、API密钥）均本地加密存储，不上传至第三方服务器，开发者不收集任何用户数据

- API对接：部分平台API可能存在调用限制或权限要求，将在开发过程中逐步适配，优先保障数据获取的合法性与稳定性

- 免责声明：本项目仅提供资产查看与管理功能，不提供任何理财建议，用户需自行判断理财风险
